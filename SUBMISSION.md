## 2. 배포 — PlayMCP in KC (둘 중 택1)

> ⚠️ PlayMCP in KC는 **예선 기간(6/15~7/14)에만** 서버 발급. 계정당 2대. 미리 발급받을 것.

- **방법 A — Git 소스**: 이 레포를 GitHub 등에 올리고, PlayMCP in KC에서 "Git 소스로 등록" → 레포의 `Dockerfile`로 자동 빌드.
- **방법 B — 컨테이너 이미지**: 로컬/CI에서 이미지를 빌드·푸시 후 "컨테이너 이미지로 등록".
  ```bash
  docker build -t <레지스트리>/legal-navigator-mcp:latest .
  docker push  <레지스트리>/legal-navigator-mcp:latest
  ```
- 포트: 서버는 `$PORT`(미설정 시 8080)를 사용. KC가 지정하는 포트 규칙에 맞출 것.
- 헬스체크: `GET /healthz` → `{"status":"ok"}`. MCP 엔드포인트: `POST /mcp`.

## 3. 등록 → 심사 → 전체공개 (공식 절차 체크리스트)

- [ ] PlayMCP in KC에서 MCP 서버 발급 → **Endpoint URL** 확보
- [ ] PlayMCP 개발자 콘솔 → "새로운 MCP 서버 등록" → 위 1번 등록값 입력
- [ ] Endpoint URL 입력 후 **"정보 불러오기" 성공 확인** (실패 시 서버 문제 — 로그 확인)
- [ ] **"임시 등록"** 저장 → "도구함에 추가" → PlayMCP AI채팅으로 테스트

- [ ] 승인 메일 수신 → 공개상태를 **"전체 공개"**로 전환
- [ ] 상세페이지 URL 복사 (예: `https://playmcp.kakao.com/mcp/xxxxx`)
- [ ] 공모전 페이지 → **"Player 예선 참여"** 비즈폼 작성·제출 *(서버 2개까지, 접수 1회)*

## 4. 비즈폼 답안 초안 (예상 항목 — 실제 폼에 맞춰 사용)

- **서비스명**: 법률 절차 길잡이
- **한줄 소개 / 주요 기능**: 위 1번 참조 + 도구 5종(절차안내·필요서류·표준서식·금액계산·법령요지)
- **타깃 사용자**: 변호사 선임은 부담스럽지만 노동 문제에 직접 대응해야 하는 직장인·아르바이트·사회초년생
- **차별점**:
  1. 정보→**행동**으로 연결(절차+서식+계산을 한 번에)
  2. 모든 법령을 **공식 출처(law.go.kr)로 grounding** → 환각 없는 정확성
  3. **변호사법 안전 설계**(개별 판단·대필 배제, 면책 고지) — 카카오톡 실서비스 노출에 적합
- **사용 예시 대화**:
  > 사용자: "3개월째 월급 250만원을 못 받았어요. 어떻게 하죠?"
  > 길잡이: 임금체불 절차(관할 노동청·소멸시효 3년·온라인 진정) 안내 → 체불임금 계산(750만원) → 진정서 표준서식 제공 → 증거 체크리스트.

## 5. 심사 기준 대응 (창의성·편의성·안정성)

- **창의성**: 법률 정보 비대칭을 해소해, 법을 모르는 사람을 *실제 실행*까지 데려가는 에이전트.
- **편의성**: 카카오톡 대화만으로 절차·서식·계산을 즉시 — 별도 앱·검색 불필요.
- **안정성**: 공식 법령 deep-link grounding, 결정론적 계산, stateless·인메모리 빠른 응답(평균<100ms), 모든 응답 면책 고지, 변호사법 경계 준수(BOUNDARIES.md).
