## 2. 배포 — PlayMCP in KC (둘 중 택1)

> ⚠️ PlayMCP in KC는 **예선 기간(6/15~7/14)에만** 서버 발급. 계정당 2대. 미리 발급받을 것.

- **방법 A — Git 소스**: 이 레포를 GitHub 등에 올리고, PlayMCP in KC에서 "Git 소스로 등록" → 레포의 `Dockerfile`로 자동 빌드.
- **방법 B — 컨테이너 이미지**: 로컬/CI에서 이미지를 빌드·푸시 후 "컨테이너 이미지로 등록".
  ```bash
  docker build -t <레지스트리>/legal-navigator-mcp:latest .
  docker push  <레지스트리>/legal-navigator-mcp:latest
  ```
- 포트: 서버는 `$PORT`(미설정 시 8080)를 사용. KC가 지정하는 포트 규칙에 맞출 것.
- 헬스체크: `GET /healthz` → `{"status":"ok"}`. MCP 엔드포인트: `POST /mcp`.

## 4. 비즈폼 답안 초안 (예상 항목 — 실제 폼에 맞춰 사용)

- **서비스명**: 법률 절차 길잡이
- **한줄 소개 / 주요 기능**: 위 1번 참조 + 도구 15종(빠른진단·자연어검색·주제목록·절차안내·필요서류·표준서식·판례조회·인용검증·시점법·법령요지·금액계산·소송비용계산·기한계산·무료법률지원연결(신청절차·준비서류)·증빙서류발급안내)
- **타깃 사용자**: 변호사 선임은 부담스럽지만 일상 법률 문제(임대차·돈거래·해고·사기·이혼/상속·교통사고 등)에 직접 대응해야 하는 일반인
- **차별점**:
  1. 정보→**행동**으로 연결(자연어 진단→절차+서식+판례+계산을 한 번에)
  2. **36개 분야 168개 주제 + 검증된 판례 187건**(전부 law.go.kr·casenote 실열람 검증, 없는 판례는 지어내지 않음)
  3. 해외 법률 AI의 약점인 **인용 환각을 `verify_citation`으로 정면 차단** + `law_updates`로 시점법 안내
  4. **변호사법 안전 설계(declaw)**(개별 판단·대필 배제, 출처 링크·면책 고지 자동) — 카카오톡 실서비스 노출에 적합
- **사용 예시 대화**:
  > 사용자: "어제 보이스피싱으로 500만원을 보냈어요."
  > 길잡이: (triage) 골든타임 즉시 행동 — 112·1332·은행 지급정지, 1394 신고 → 피해금환급절차 안내 → 증거 체크리스트 → 관련 판례·계좌명의인 책임 안내.

## 5. 심사 기준 대응 (창의성·편의성·안정성)

- **창의성**: 법률 정보 비대칭을 해소해, 법을 모르는 사람을 *실제 실행*까지 데려가는 에이전트. 36개 분야·168주제·187판례 + 인용검증·시점법·트리아지로 "변호사 없이 혼자 끝까지".
- **편의성**: 카카오톡 대화만으로 자연어 진단→절차·서식·판례·계산을 즉시 — 별도 앱·검색 불필요.
- **안정성**: 공식 법령 deep-link grounding + **검증된 판례만 수록(환각 차단, `verify_citation`으로 인용 진위·유효성 확인)**, 결정론적 계산, stateless·인메모리 빠른 응답(평균<100ms), 모든 응답 출처·면책 고지, 변호사법 경계 준수(BOUNDARIES.md).
