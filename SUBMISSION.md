> ⚠️ PlayMCP in KC는 **예선 기간(6/15~7/14)에만** 서버 발급. 계정당 2대. 미리 발급받을 것.

- **방법 B — 컨테이너 이미지**: 로컬/CI에서 이미지를 빌드·푸시 후 "컨테이너 이미지로 등록".
  ```bash
  docker build -t <레지스트리>/legal-navigator-mcp:latest .
  docker push  <레지스트리>/legal-navigator-mcp:latest
  ```
- 포트: 서버는 `$PORT`(미설정 시 8080)를 사용. KC가 지정하는 포트 규칙에 맞출 것.

## 5. 심사 기준 대응 (창의성·편의성·안정성)

- **창의성**: 법률 정보 비대칭을 해소해, 법을 모르는 사람을 *실제 실행*까지 데려가는 에이전트. 49개 분야·203주제·197판례·법률용어 125(외국인노동자·이주여성·청소년·장애인·북한이탈주민·플랫폼노동·국가유공자·기초생활·한부모·농어업인·노인·정신건강·범죄피해자·자살예방/유족·재난안전 등 취약계층·위기 35주제 포함) + 인용검증·시점법·트리아지로 "변호사 없이 혼자 끝까지".
- **편의성**: 카카오톡 대화만으로 자연어 진단→절차·서식·판례·계산을 즉시 — 별도 앱·검색 불필요. 표준 서식은 응답에 **공식 양식 받는 곳**과 **`.txt` 다운로드 링크**가 함께 붙어, 받은 파일을 구글 드라이브·카카오톡 '나에게 보내기'·메일로 바로 보관·공유.
- **안정성**: 공식 법령 deep-link grounding + **검증된 판례만 수록(환각 차단, `verify_citation`으로 인용 진위·유효성 확인)**, 결정론적 계산, stateless·인메모리 빠른 응답(평균<100ms), 모든 응답 출처·면책 고지, 변호사법 경계 준수(BOUNDARIES.md). **자동 테스트 28개(vitest — 계산 결정성·데이터 정합성 불변식·인용 거짓확인 회귀·통합)로 회귀 방지**, 도메인 병합은 키 충돌 시 즉시 실패(mergeStrict)해 데이터 유실을 차단.
